# Git Documentation Repository

## Overview
This repository provides comprehensive documentation on various Git topics, intended to help both beginners and experienced users. It covers essential concepts, best practices, and advanced techniques, with each topic organized into separate markdown files for clarity and ease of navigation.

## Topics Covered
1. **SSH Key Configuration** (`ssh-key.md`):
    - How to set up and manage SSH keys for secure authentication with GitHub.
    - Detailed steps for generating new keys, adding them to `ssh-agent`, and configuring them on GitHub.
2. **Git Basics** (`git.md`) - _coming soon_:
    - Introduction to Git, version control, and basic commands.
    - Working with repositories, branches, and commits.
